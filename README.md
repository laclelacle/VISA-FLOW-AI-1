`index.html`을 더블클릭해서 브라우저로 열면 바로 실행됩니다.

## 주요 기능

- 목적(여행/워킹홀리데이/해외취업/유학) × 국가별 맞춤 비자 안내
- 국가 비교 기능 (최대 3개국)
- AI 챗봇 "비비(BIBI)" — 국가·목적별 실데이터 기반 Q&A
- 국가 검색 자동완성 + 인기 검색어
- 실시간 공지사항 (최근 3개월 실제 공지 반영)
- D-DAY 출국 준비 타임라인 + 이메일 발송(EmailJS 연동 시 자동 발송)
- 제출서류 · 공식 서식 모음 페이지

## 이메일 자동발송 활성화 (선택)

`index.html` 상단 부근에서 아래 세 값을 채우면 D-DAY 일정 이메일이 실제로 자동 발송됩니다.
비워두면 대신 메일 작성창(mailto)이 열립니다.

```js
const EMAILJS_SERVICE_ID='';   // emailjs.com에서 발급
const EMAILJS_TEMPLATE_ID='';
const EMAILJS_PUBLIC_KEY='';
```
